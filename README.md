Pokédex Iallen

Uma aplicação Pokédex estilo Pokémon GO construída com React Native.

Funcionalidades

- Lista de Pokémons com scroll infinito
- Detalhes completos de cada Pokémon
- Design inspirado no Pokémon GO
- Modal com estatísticas e informações
- Interface responsiva

Tecnologias

- React Native
- Expo
- PokeAPI
- JavaScript

Como Usar

1. Baixe todos os arquivos
2. Execute `npm install` para instalar as dependências
3. Execute `npm start` para rodar o projeto

Código

O código principal está no arquivo `App.js`

Objetivo

Projeto educativo para demonstrar consumo de API (POKEAPI) e desenvolvimento React Native.

Desenvolvido como atividade acadêmica.
